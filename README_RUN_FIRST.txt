MEP Progress Tracker - Portable HTML Version
===========================================

This version is designed to open on other computers without installation.

Start Here
----------
1. Extract the ZIP file or clone the repository.
2. Double-click MEP_Progress_Tracker_Portable.html.
3. Read QUICK_START.txt for basic use.
4. Read USER_MANUAL.txt for the full English user manual.

No requirements:
- No Python
- No .NET
- No BAT
- No HTA
- No admin rights

Main features:
- Floor-wise MEP progress tracking
- System / sub-system selection
- Planned % and Actual %
- Variance calculation
- Completed / Delayed / In Progress / Not Started status
- Generate sample floors
- Save/load project data as JSON
- Export CSV
- Export Excel-compatible .XLS
